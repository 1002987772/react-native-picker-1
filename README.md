# react-native-pickers
纯JS实现的React-Native 日期选择控件、地址选择控件等。

时间不多，肯定有bug，欢迎留言第一时间维护。

由于很多属性并没有封装好开放，所以暂时直接以源码形式分享，未插件化（2018年3月5日）

欢迎直接下载代码删改使用，欢迎沟通了解实现原理，有帮得上的，记得给星哦~
